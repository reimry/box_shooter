git init (initialize a new repository)

git status (check which files are tracked or changed)

git add . (stage all changes for a commit)

git commit -m "your message" (save staged changes with a description)

git log --oneline (view a condensed history of your commits)

git branch <branch-name> (create a new feature branch)

git checkout <branch-name> (switch to a specific branch)

git merge <branch-name> (combine a completed feature into your main branch)

git remote add origin <url> (link your local repo to GitHub/GitLab)

git push -u origin main (upload your code to the server for the first time)
